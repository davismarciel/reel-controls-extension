# Reel Controls

Extensao para Firefox que adiciona controles de progresso e volume aos Reels do Instagram.

## Recursos

- Barra de progresso do video
- Controle de volume
- Botao de mute/unmute
- Configuracoes em pagina de opcoes

## Como testar localmente no Firefox

1. Abra `about:debugging#/runtime/this-firefox`
2. Clique em `Load Temporary Add-on...`
3. Selecione o arquivo `manifest.json` desta pasta
4. Acesse o Instagram e abra um Reel

## Configuracoes

Abra as opcoes da extensao para ajustar:

- Ativar/desativar no Instagram
- Volume inicial
- Tempo para ocultar os controles

## Privacidade

A extensao nao coleta, armazena ou envia dados pessoais.
